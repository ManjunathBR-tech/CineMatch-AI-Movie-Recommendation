# 🎬 CineMatch - AI Movie Recommendation System

CineMatch is an AI-powered movie recommendation system built using Python, Tkinter, Scikit-learn, and MySQL.

## 🚀 Features

- 🔐 Login & Register System
- 🎥 AI Movie Recommendations
- ❤️ Save Favorite Movies
- 📊 Analytics Dashboard
- 🖼️ TMDB Movie Posters
- 🌌 Modern Netflix-style GUI

## 🛠️ Technologies Used

- Python
- Tkinter
- ttkbootstrap
- Pandas
- Scikit-learn
- MySQL
- TMDB API
- Matplotlib

## ▶️ Run Project

```bash
python main.py
